# Promo video — phone shot list

The draft video ([social_pressure_promo_draft.mp4](social_pressure_promo_draft.mp4))
has a dark placeholder card at ~0:42. Your phone footage replaces it. Everything
else is done — record these clips, drop them in `promo/phone_clips/`, and the
final video gets rebuilt with them spliced in and captioned.

## Setup (2 minutes)

- Use your Android phone's built-in screen recorder (swipe down → Screen record).
- Turn on **Do Not Disturb** first so stray notifications don't photobomb —
  but allow the Social Pressure test ping through (or just briefly toggle DND
  off for clip 1, since the ping *is* the shot).
- Portrait orientation. Don't rush the taps — a beat of stillness after each
  screen makes editing easier. Everything gets trimmed, so long is fine.
- Use the live app: https://social-pressure-web.onrender.com with an account
  that's in at least one project (the notification copy reads better with a
  partner project visible).

## Clip 1 — enable notifications + the ping (~20s raw)

1. Start on a project screen showing the "Get a ping when your partners
   check in" banner (or Profile → notification controls if the banner is
   dismissed).
2. Tap **Enable** → Chrome's permission dialog appears → tap **Allow**.
3. Go to Profile → tap **Send test notification**.
4. Pull down the notification shade so the Social Pressure ping is clearly
   visible for ~2 seconds.

## Clip 2 — install the app (~15s raw)

1. In Chrome on the app, open the **⋮ menu**.
2. Tap **Add to Home screen** → confirm **Install**.
3. Go to the home screen, show the Social Pressure icon for a beat.
4. Tap the icon — the app opens full-screen (no browser bar). Hold ~2s.

## Clip 3 (optional) — system notification settings (~10s raw)

Only if you want the "check your system settings" beat:

1. Android **Settings → Apps → Social Pressure → Notifications** (installed
   PWA) — show the toggle ON.
2. Or for browser-only users: Settings → Apps → Chrome → Notifications.

## When you're done

Drop the file(s) into `promo/phone_clips/` (any filename, mp4/webm) and say the
word — they get trimmed, speed-matched, captioned to match the rest, and the
final ~60s video is rebuilt.
